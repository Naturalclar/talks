### react-native

- Cross Platform 開発が出来るフレームワーク
  - iOS、Android 等
- react を知っていれば似たような記法でかける。
  - `<div>`や`<p>`などは使わない
  - `<View>`や`<Text>`など独自のタグを使う

<CodeSurferLayout>

```jsx subtitle="StyleSheet Example"
import React from "react-native";
import { View, Text, StyleSheet } from "react-native";

type Props = {
  isActive: boolean
};

export const Title = ({ isActive }: Props) => (
  <View style={styles.container}>
    <Text style={[styles.title, isActive && styles.activeTitle]} />
  </View>
);

const styles = StyleSheet.create({
  container: {
    borderRadius: 4,
    borderWidth: 0.5,
    borderColor: "#d6d7da"
  },
  title: {
    fontSize: 19,
    fontWeight: "bold"
  },
  activeTitle: {
    color: "red"
  }
});
```

## react-native と Web の差

---

### react-native と Web の差

- Display は基本 `flex`
- flexDirection はデフォルトで`column`
- ショートハンドは使えない
- flex の微妙な差異

---

## Display は基本 `flex`

- react-native は[yoga](https://github.com/facebook/yoga)を使って View を作成している
- yoga は Cross Platform の FlexBox を実装するライブラリ

---
