## React Native Community に入った経緯

---

<Header>きっかけ</Header>

- 海外転職の為に本職以外の活動(OSS)がしたかった
- 前職の Tech Lead (Node JS Collaborator)が辞めた
- 新しく技術部の柱になるチャンス
- 社内で React Native を使っていた

<Notes>
  そもそも自分がなぜ React Native Community に入ろうと思ったかについて話します。
  自分は今日本でエンジニアとして活動しているのですが、そもそも日本に来た理由が日本の方が未経験エンジニアとして採用されやすいと感じたから。
  いつかは海外に戻りたいと思っていたので、日本で活動しつつ、海外でもアピールポイントになりそうなOSS活動に興味を持っていました。
  そんな中、前職でOSS活動を行っていたTech
  Leadが辞めるというイベントが発生しました。 その辞めたTech LeadがNode
  JSのコラボレーターでもあり、社内の技術勉強会が彼中心に回っていました。
  これはチャンスだなと思い、自分がその代わりになって何かしら技術の柱になろうと思いました。
  そして、社内でReact Nativeを使っていたので、React
  NativeにContributeしようと思ったのがきっかけです。
</Notes>

<Header>Shopify</Header>

- [FlashList](https://shopify.github.io/flash-list/)
- [React Native Skia](https://github.com/Shopify/react-native-skia)

<Notes>
  Shopify は React Native での開発をサポートする OSS を作っている。 FlashList は
  React Native の FlatList の高速化を目的としたライブラリです。 React Native
  Skia は React Native で Skia を使うためのライブラリです。
</Notes>

---

<Header>Software Mansion</Header>

- [React Native Gesture Handler]()
- [React Native Reanimated]()
- [React Native Screens]()

<Notes>

Software Mansion は React Native での開発をサポートする OSS を作っている。
React Native Gesture Handler は React Native
でのジェスチャーを扱うためのライブラリです。 React Native Reanimated は React
Native でアニメーションを扱うためのライブラリです。 React Native Screens は
React Native でスクリーン遷移を高速化するためのライブラリです。

</Notes>
