## Code Surfer とは

- [これ](https://github.com/pomber/code-surfer)
- このスライドを作ってる`mdx-deck`と使えるライブラリ
- コードを深堀りしたい時に便利

<CodeSurferLayout>

```js title="タイトルつけれる"
function hello(name) {
  console.log("hello " + name);
}

hello("world");
```

```js 1:3 subtitle="字幕も今まで通りつけれる"
function hello(name) {
  console.log("hello " + name);
}

hello("world");
```

```js subtitle="前のページとのDiffを取って、新しく追加されたコードを挿入"
function helloFactory() {
  return name => {
    console.log("hello " + name);
  };
}

function hello(name) {
  console.log("hello " + name);
}

hello("world");
```

```js subtitle="変更があった箇所は差し替えられる"
function helloFactory() {
  return name => {
    console.log("こんにちは " + name);
  };
}

function hello(name) {
  console.log("こんにちは " + name);
}

hello("world");
```

```js 1:7 subtitle="削除されたところは消える"
function helloFactory() {
  return name => {
    console.log("こんにちは " + name);
  };
}

hello("world");
```
